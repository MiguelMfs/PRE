# Programação Estruturada

## Descrição

Este repositório contém uma coleção de exercícios desenvolvidos na disciplina **Programação Estruturada**.

Os códigos abordam conceitos fundamentais da linguagem C, incluindo:

- Declaração e utilização de funções;
- Passagem de parâmetros por valor e por referência;
- Ponteiros e aritmética de ponteiros;
- Vetores e matrizes;
- Estruturas (`struct`);
- Alocação dinâmica de memória;
- Uso de `malloc`, `calloc` e `realloc`;
- Manipulação de arquivos de dados em memória;
- Organização e processamento de informações utilizando registros.

## Conteúdos abordados

### Funções
Exercícios envolvendo criação e chamada de funções, retorno de valores e manipulação de parâmetros.

### Ponteiros
Implementações utilizando ponteiros para:
- Alteração de valores em funções;
- Manipulação de vetores;
- Acesso a endereços de memória;
- Operações utilizando aritmética de ponteiros.

### Vetores e Matrizes
Programas envolvendo:
- Leitura e impressão de vetores;
- Operações matemáticas;
- Busca de valores;
- Manipulação de matrizes dinâmicas.

### Estruturas (`struct`)
Exercícios utilizando registros para armazenar informações como:
- Alunos;
- Produtos;
- Funcionários;
- Cidades;
- Livros;
- Compromissos;
- Agendas.

### Alocação Dinâmica
Exercícios utilizando gerenciamento de memória com:
- `malloc()` para reserva de espaço;
- `calloc()` para inicialização de memória;
- `realloc()` para expansão de vetores;
- `free()` para liberação de memória.

## Objetivo

Desenvolver a capacidade de criar programas estruturados em C, aplicando boas práticas de programação, organização de código e gerenciamento eficiente de memória.

## Tecnologias utilizadas

- Linguagem C
- Compilador C (GCC ou equivalente)
- Ambiente de desenvolvimento compatível com C

## Estrutura dos códigos

Cada exercício possui seu próprio programa independente, contendo:

- Declaração das variáveis e estruturas necessárias;
- Funções auxiliares quando solicitadas;
- Função principal `main()`;
- Entrada de dados pelo usuário;
- Processamento das informações;
- Exibição dos resultados.
